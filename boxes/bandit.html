Bandit 5

Achar um arquivo no meio de muitas pastas, usar o comando find na propria pasta, setando o tipo para -f, significando que é um arquivo regular, no caso do 06 era um arquivo readable, entao -readable, e tinhamos o tamanho do arquivo, que era de 1033 bytes, logo -size 1033c. c no final pois é para o número exato de bytes, caso quisesse setar para blocos de bytes, seria -size 512b para blocos de 512 bytes cada por exemplo. Por fim, como era um arquivo nao executavel, ! -executable para negar que era executavel.

comando final:

find . -type f -readable -size 1033c ! -executable

Similar ao nivel anterior, só que nesse é necessário procurar no servidor inteiro, tendo apenas o tamanho do arquivo e os grupos que sao donos dele.

para usuario -user, para grupo -group, e como era no servidor inteiro, o diretório de partida é o root (/)
